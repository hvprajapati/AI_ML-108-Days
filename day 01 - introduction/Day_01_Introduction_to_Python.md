# Theory Notes

## What is Programming?

Programming means writing instructions for a computer.

Example:

* Open Browser
* Search Google
* Play YouTube

Computers execute instructions exactly as written.

---

## What is Python?

Python is a high-level programming language created by Guido van Rossum in 1991.

Features:

* Easy to Learn
* Easy to Read
* Platform Independent
* Open Source

---


---

## Program 4 - Student Information Card

```python
name = input("Enter Name: ")
college = input("Enter College: ")
branch = input("Enter Branch: ")

print("\n----- Student Card -----")
print("Name:", name)
print("College:", college)
print("Branch:", branch)
```

---

## Program 5 - Simple Calculator

```python
num1 = int(input("Enter First Number: "))
num2 = int(input("Enter Second Number: "))

print("Addition =", num1 + num2)
print("Subtraction =", num1 - num2)
print("Multiplication =", num1 * num2)
print("Division =", num1 / num2)
```

---

## Program 6 - Rectangle Area Calculator

```python
length = float(input("Length: "))
width = float(input("Width: "))

area = length * width

print("Area =", area)
```

# Common Student Mistakes

### Missing Quotes

Wrong:

```python
print(Hello)
```

Correct:

```python
print("Hello")
```

### Forgetting Brackets

Wrong:

```python
print "Hello"
```

Correct:

```python
print("Hello")
```
